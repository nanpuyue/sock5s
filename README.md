# A Simple Socks5 Proxy Server

## Usage

```
sock5s 0.2.1
南浦月 <nanpuyue@gmail.com>
A Simple Socks5 Proxy Server

USAGE:
    sock5s --listen <LISTEN ADDR>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -l, --listen <LISTEN ADDR>    Specify the listen addr
```
